LgScript

A custom script module for Thief, Thief 2, and System Shock 2. With it you can
create your own scripts using the Lua language.

The script documentation for fan-mission authors is hosted online at The
Dromesday Book. Please go to https://dromed.whoopdedo.org/lgscript/index to
learn how you can use custom scripts in your missions.
